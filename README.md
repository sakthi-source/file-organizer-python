# File Organizer (Python)

This project automatically organizes files into folders based on file types.

## Features
- Sorts images, documents, and music files
- Creates folders automatically
- Moves files into correct folders

## Technologies Used
- Python
- os module
- shutil module
- This project was built as part of learning Python and automation.
